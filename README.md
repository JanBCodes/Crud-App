# Movie and TV Streaming Service - CRUD Demo

## Techologies & Libraries Used/Will Be
> MongoDB
> Node.js
> Express.js
> Mongoose
> Nodemon
> Dotenv
> Express-FileUpload
> UUID
> Path
> Bcryptjs
